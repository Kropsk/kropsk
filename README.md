## About

I'm a backend-focused developer working primarily with **game infrastructure**, **high-throughput JVM systems**, and **containerized services**.

Most of my experience comes from building and maintaining **large Minecraft server networks**, dealing with real-world constraints such as memory pressure, GC behavior, network latency, and horizontal scalability.

<img src="https://media.tenor.com/KTS9iInOC20AAAAi/star-aesthetic.gif" width="100%">

## What I Work With

- Large-scale **Folia / Paper** server architectures  
- JVM performance tuning (memory, GC, OOM mitigation)  
- Redis-based caching and data synchronization  
- Backend services with **Quarkus**  
- Dockerized environments on Linux  
- Proxy and network-level orchestration (Velocity, Geyser)
